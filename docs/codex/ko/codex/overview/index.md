# Codex

Source URL: https://developers.openai.com/codex/overview

Codex는 소프트웨어 개발을 위한 OpenAI의 코딩 에이전트입니다. ChatGPT Plus, Pro, Business, Edu, 그리고 Enterprise 플랜에 Codex가 포함되어 있습니다. Codex는 다음과 같은 작업을 도와줄 수 있습니다:

  * **코드 작성**: 만들고 싶은 것을 설명하면 Codex가 기존 프로젝트 구조와 규칙에 맞춰 의도에 부합하는 코드를 생성합니다.

  * **낯선 코드베이스 이해**: Codex는 복잡하거나 레거시 코드도 읽고 설명해 주어, 팀이 시스템을 어떤 방식으로 구성하는지 파악할 수 있게 도와줍니다.

  * **코드 리뷰**: Codex가 코드를 분석해 잠재적인 버그, 논리 오류, 처리되지 않은 에지 케이스를 식별합니다.

  * **디버그 및 문제 해결**: 문제가 발생하면 Codex가 실패 지점을 추적하고 근본 원인을 진단하며 타깃팅된 수정안을 제안합니다.

  * **개발 작업 자동화**: Codex는 리팩터링, 테스트, 마이그레이션, 설정 작업과 같은 반복적 워크플로를 실행하여, 엔지니어는 더 높은 수준의 작업에 집중할 수 있습니다.
