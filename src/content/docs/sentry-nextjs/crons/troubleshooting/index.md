# 문제 해결 | Next.js용 Sentry

반복 작업 오류가 모니터 상세 페이지에 표시되지 않는 이유는 무엇인가요?

[오류를 모니터에 연결](https://docs.sentry.io/platforms/javascript/guides/nextjs/crons.md#connecting-errors-to-cron-monitors)하지 않았을 수 있습니다.

모니터 실패 시 알림을 받지 못하는 이유는 무엇인가요?

[모니터 알림을 설정](https://docs.sentry.io/platforms/javascript/guides/nextjs/crons.md#alerts)하지 않았을 수 있습니다.

check-in에 대한 crons 데이터 보존 정책은 무엇인가요?

현재 데이터 보존 정책은 90일입니다.

6필드 crontab 표현식을 사용하는 모니터 스케줄을 지원하나요?

현재는 5필드 crontab 표현식만 지원합니다.
