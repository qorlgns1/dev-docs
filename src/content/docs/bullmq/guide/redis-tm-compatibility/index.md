# Redis™ 호환성

Redis에는 여러 대안이 있으며 BullMQ는 6.2.0 이상 버전과 완전히 Redis™ 호환되지만, 모든 대안이 제대로 동작하는 것은 아닙니다. 이 섹션에서는 BullMQ를 공식적으로 지원하고, 호환성이 계속 유지되는지 확인하기 위해 저희가 정기적으로 테스트하는 벤더를 소개합니다.
