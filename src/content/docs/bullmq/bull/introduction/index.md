# 소개

Bull은 BullMQ의 레거시 버전입니다. 현재도 여전히 널리 사용되고 있기 때문에 새로운 주요 기능 추가는 하지 않지만 버그 수정은 계속 유지보수됩니다. 검증된 큐 라이브러리를 사용하고 싶고 더 나은 TypeScript 통합이나 최신 기능이 꼭 필요하지 않다면, 앞으로도 오랫동안 이 라이브러리를 문제없이 사용할 수 있습니다.

### 사용 중인 곳

Bull은 오랫동안 NodeJS 생태계의 일부였으며, 상용 및 오픈소스 프로젝트 모두에서 많은 조직이 사용하고 있습니다. 몇 가지 대표 사례는 다음과 같습니다.
