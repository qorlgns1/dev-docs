# Nuxt

Prisma ORM을 Nuxt 애플리케이션에 통합하는 방법을 알아보세요.

`@prisma/nuxt` 모듈은 더 이상 사용되지 않으며 권장되지 않습니다. Nuxt 애플리케이션에 Prisma ORM을 통합하는 권장 방식은 [Nuxt guide](https://docs.prisma.io/docs/guides/frameworks/nuxt)를 따라 주세요.
