# dir

- **타입:** `string`
- **CLI:** `--dir=<path>`
- **기본값:** `root`와 동일

테스트 파일을 스캔할 기본 디렉터리입니다. root가 전체 프로젝트를 포함하는 경우, 이 옵션을 지정해 테스트 검색 속도를 높일 수 있습니다.
