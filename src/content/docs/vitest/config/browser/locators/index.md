# browser.locators

내장 [browser locators](https://vitest.dev/api/browser/locators)를 위한 옵션입니다.

## browser.locators.testIdAttribute

- **Type:** `string`
- **Default:** `data-testid`

`getByTestId` locator로 요소를 찾는 데 사용되는 속성입니다.
