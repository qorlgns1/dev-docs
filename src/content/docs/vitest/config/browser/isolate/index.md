# browser.isolate

- **Type:** `boolean`
- **Default:** [`--isolate`](https://vitest.dev/config/#isolate)와 동일
- **CLI:** `--browser.isolate`, `--browser.isolate=false`

모든 테스트를 별도의 iframe에서 실행합니다.

::: danger DEPRECATED
이 옵션은 더 이상 사용되지 않습니다. 대신 [`isolate`](https://vitest.dev/config/#isolate)를 사용하세요.
:::
