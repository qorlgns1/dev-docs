# silent {#silent}

- **Type:** `boolean | 'passed-only'`
- **Default:** `false`
- **CLI:** `--silent`, `--silent=false`

테스트의 콘솔 출력을 숨깁니다.

실패한 테스트의 로그만 보려면 `'passed-only'`를 사용하세요. 실패한 테스트의 로그는 테스트가 완료된 후 출력됩니다.
