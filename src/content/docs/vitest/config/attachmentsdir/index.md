# attachmentsDir

- **유형:** `string`
- **기본값:** `'.vitest-attachments'`

프로젝트 루트를 기준으로 [`context.annotate`](https://vitest.dev/guide/test-context#annotate)로 생성된 첨부 파일을 저장할 디렉터리 경로입니다.
