# slowTestThreshold

- **유형**: `number`
- **기본값**: `300`
- **CLI**: `--slow-test-threshold=<number>`, `--slowTestThreshold=<number>`

테스트 또는 스위트가 느린 것으로 간주되어 결과에 그렇게 보고되는 기준이 되는 밀리초 단위 시간입니다.
