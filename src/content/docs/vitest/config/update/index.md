# update {#update}

- **유형:** `boolean`
- **기본값:** `false`
- **CLI:** `-u`, `--update`, `--update=false`

스냅샷 파일을 업데이트합니다. 변경된 모든 스냅샷을 업데이트하고, 더 이상 사용되지 않는 스냅샷은 삭제합니다.
