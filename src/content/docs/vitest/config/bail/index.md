# bail

- **Type:** `number`
- **Default:** `0`
- **CLI**: `--bail=<value>`

지정한 수의 테스트가 실패하면 테스트 실행을 중지합니다.

기본적으로 Vitest는 일부 테스트가 실패하더라도 모든 테스트 케이스를 실행합니다. 하지만 이는 100% 성공한 빌드에만 관심이 있고, 테스트 실패가 발생하면 가능한 한 빨리 실행을 중단하고 싶은 CI 빌드에서는 바람직하지 않을 수 있습니다. `bail` 옵션을 사용하면 실패가 발생한 뒤 추가 테스트 실행을 막아 CI 실행 속도를 높일 수 있습니다.
