# snapshotSerializers

- **유형:** `string[]`
- **기본값:** `[]`

스냅샷 테스트를 위한 스냅샷 serializer 모듈 경로 목록입니다. 커스텀 스냅샷 serializer를 추가하려는 경우 유용합니다. 자세한 내용은 [Custom Serializer](https://vitest.dev/guide/snapshot#custom-serializer)를 참고하세요.
