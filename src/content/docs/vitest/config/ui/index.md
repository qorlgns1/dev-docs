# ui

- **타입:** `boolean`
- **기본값:** `false`
- **CLI:** `--ui`, `--ui=false`

[Vitest UI](https://vitest.dev/guide/ui)를 활성화합니다.

::: warning
이 기능을 사용하려면 [`@vitest/ui`](https://www.npmjs.com/package/@vitest/ui) 패키지가 설치되어 있어야 합니다. 아직 설치되어 있지 않다면, 처음으로 테스트 명령을 실행할 때 Vitest가 이를 설치합니다.
:::
