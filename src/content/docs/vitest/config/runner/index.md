# runner

- **Type**: `VitestRunnerConstructor`
- **Default**: 테스트 실행 시 `node`, 벤치마크 실행 시 `benchmark`

커스텀 테스트 러너의 경로입니다. 이 기능은 고급 기능이며, 커스텀 라이브러리 러너와 함께 사용해야 합니다. 자세한 내용은 [문서](https://vitest.dev/api/advanced/runner)에서 확인할 수 있습니다.
