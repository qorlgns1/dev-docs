# Next.js 국제화(i18n)

`next-intl`은 Next.js를 위한 국제화 툴킷으로, 다음을 도와줍니다:

  1. 현지화된 번역 렌더링
  2. 날짜, 숫자 등 포맷팅
  3. 국제화 라우팅 처리

… 현지화된 앱을 구축하기 위한 필수 기반을 제공합니다.

[![동영상 미리보기](https://next-intl.dev/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Fthumb.06bfa870.jpg&w=384&q=75)](https://learn.next-intl.dev)

### 국제화는 단순히 단어를 번역하는 것이 아닙니다

실제 프로젝트를 통해 기초부터 고급 패턴까지, Next.js로 매력적인 다국어 앱을 만드는 방법을 배워보세요.

[시작하기→](https://learn.next-intl.dev)

[예제 앱](https://next-intl.dev/examples)을 살펴보며 `next-intl`로 무엇을 만들 수 있는지 더 알아보세요.
