# 환경

`next-intl` API는 다음 환경에서 사용할 수 있습니다:

[서버 및 클라이언트 컴포넌트→](https://next-intl.dev/docs/environments/server-client-components)[서버 액션, 메타데이터 및 라우트 핸들러→](https://next-intl.dev/docs/environments/actions-metadata-route-handlers)[에러 파일(예: not-found)→](https://next-intl.dev/docs/environments/error-files)[테스팅→](https://next-intl.dev/docs/environments/testing)[코어 라이브러리(agnostic)→](https://next-intl.dev/docs/environments/core-library)

최신 브라우저와 서버 런타임은 일반적으로 `next-intl`에 필요한 모든 JavaScript API를 지원하지만, [런타임 요구 사항](https://next-intl.dev/docs/environments/runtime-requirements)을 통해 다시 확인할 수 있습니다.
