# Next.js 커뮤니티

마지막 업데이트 2026년 2월 20일

주간 500만 건 이상의 다운로드를 기록하는 Next.js는 전 세계 개발자들이 활발히 활동하는 커뮤니티를 보유하고 있습니다. 커뮤니티에 참여하는 방법은 다음과 같습니다:

## Contributing[](https://nextjs.org/docs/community#contributing)

Next.js 개발에 기여할 수 있는 몇 가지 방법이 있습니다:

  * [문서](https://nextjs.org/docs/community/contribution-guide): Next.js 사용 방법을 이해하는 데 도움이 되는 개선 의견을 제안하거나 새로운 섹션을 작성하세요.
  * [예제](https://github.com/vercel/next.js/tree/canary/examples): 새로운 예제를 만들거나 기존 예제를 개선하여 Next.js를 다른 도구와 서비스에 통합하도록 돕습니다.
  * [코드베이스](https://github.com/vercel/next.js/tree/canary/contributing/core): 기본 아키텍처를 더 자세히 알아보고 버그 수정, 오류 해결, 신규 기능 제안을 진행하세요.

## Discussions[](https://nextjs.org/docs/community#discussions)

Next.js에 대한 질문이 있거나 다른 사람을 돕고 싶다면 언제든지 대화에 참여하세요:

  * [GitHub Discussions](https://github.com/vercel/next.js/discussions)
  * [Discord](https://discord.com/invite/bUG2bvbtHy)
  * [Reddit](https://www.reddit.com/r/nextjs)

## Social Media[](https://nextjs.org/docs/community#social-media)

최신 업데이트는 [Twitter](https://x.com/nextjs)에서 확인하고, Next.js 영상을 보려면 [Vercel YouTube 채널](https://www.youtube.com/@VercelHQ)을 구독하세요.

## Code of Conduct[](https://nextjs.org/docs/community#code-of-conduct)

포용적이고 환영받는 커뮤니티를 만들고자 합니다. 따라서 모든 구성원이 우리의 [행동 강령](https://github.com/vercel/next.js/blob/canary/CODE_OF_CONDUCT.md)을 준수해 주시길 요청드립니다. 이 문서는 참가자 행동에 대한 기대를 명시합니다. 꼭 읽어 보시고 안전하고 존중하는 환경을 유지하는 데 함께해 주세요.

- [기여 가이드](https://nextjs.org/docs/community/contribution-guide)
  - Contribution GuideNext.js 문서에 기여하는 방법을 알아보세요

- [Rspack](https://nextjs.org/docs/community/rspack)
  - `next-rspack` 플러그인을 사용하여 Next.js를 Rspack으로 번들링하세요.
