# 업그레이드

마지막 업데이트 2026년 2월 20일

버전별 가이드를 따라 최신 Next.js 버전으로 업그레이드하는 방법을 알아보세요.

- [Codemods](https://nextjs.org/docs/pages/guides/upgrading/codemods)
  - 코드모드새 기능이 출시될 때 코드모드로 Next.js 코드베이스를 업그레이드하세요.

- [버전 10](https://nextjs.org/docs/pages/guides/upgrading/version-10)
  - Next.js 애플리케이션을 버전 9에서 10으로 업그레이드하세요.

- [버전 11](https://nextjs.org/docs/pages/guides/upgrading/version-11)
  - Next.js 애플리케이션을 버전 10에서 11로 업그레이드하세요.

- [버전 12](https://nextjs.org/docs/pages/guides/upgrading/version-12)
  - Next.js 애플리케이션을 버전 11에서 12로 업그레이드하세요.

- [버전 13](https://nextjs.org/docs/pages/guides/upgrading/version-13)
  - Next.js 애플리케이션을 버전 12에서 13으로 업그레이드하세요.

- [버전 14](https://nextjs.org/docs/pages/guides/upgrading/version-14)
  - Next.js 애플리케이션을 버전 13에서 14로 업그레이드하세요.

- [버전 9](https://nextjs.org/docs/pages/guides/upgrading/version-9)
  - Next.js 애플리케이션을 버전 8에서 9로 업그레이드하세요.
