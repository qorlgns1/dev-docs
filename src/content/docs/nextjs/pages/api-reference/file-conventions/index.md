# 파일 시스템 규칙

마지막 업데이트: 2026년 2월 20일

- [instrumentation.js](https://nextjs.org/docs/pages/api-reference/file-conventions/instrumentation)
  - 파일에 대한 API 참고서입니다.
- [프록시](https://nextjs.org/docs/pages/api-reference/file-conventions/proxy)
  - Proxy 요청이 완료되기 전에 코드를 실행하기 위해 Proxy를 사용하는 방법을 알아보세요.
- [public](https://nextjs.org/docs/pages/api-reference/file-conventions/public-folder)
  - 디렉터리에서 이미지와 같은 정적 파일을 제공하는 방법을 알아보세요.
- [src 디렉토리](https://nextjs.org/docs/pages/api-reference/file-conventions/src-folder)
  - `src` 폴더에 페이지를 저장하여 루트 `pages` 디렉터리의 대안으로 사용하는 방법을 알아보세요.
