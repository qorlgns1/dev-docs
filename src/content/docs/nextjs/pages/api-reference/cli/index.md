# CLI

마지막 업데이트: 2026년 2월 20일

Next.js에는 **두 개의** 명령줄 인터페이스(CLI) 도구가 포함되어 있습니다:

  * **`create-next-app`** : 기본 템플릿이나 공개 GitHub 저장소의 [예제](https://github.com/vercel/next.js/tree/canary/examples)를 사용해 새로운 Next.js 애플리케이션을 빠르게 생성합니다.
  * **`next`** : Next.js 개발 서버를 실행하고 애플리케이션을 빌드하는 등 다양한 작업을 수행합니다.

- [create-next-app CLI](https://nextjs.org/docs/pages/api-reference/cli/create-next-app)
  - create-next-app CLI 하나의 명령으로 Next.js 앱을 생성하세요.
- [next CLI](https://nextjs.org/docs/pages/api-reference/cli/next)
  - Next.js CLI로 애플리케이션을 실행하고 빌드하는 방법을 알아보세요.
