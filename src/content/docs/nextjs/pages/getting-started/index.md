# 시작하기 - 페이지 라우터

마지막 업데이트 2026년 2월 20일

- [설치](https://nextjs.org/docs/pages/getting-started/installation)
  - `create-next-app`으로 새로운 Next.js 애플리케이션을 만드는 방법, TypeScript·ESLint 설정, `next.config.js` 구성 방법을 알아보세요.

- [프로젝트 구조](https://nextjs.org/docs/pages/getting-started/project-structure)
  - Next.js 프로젝트의 폴더 및 파일 규칙과 프로젝트 구성 방법을 학습하세요.

- [Images](https://nextjs.org/docs/pages/getting-started/images)
  - 이미지내장된 `next/image` 컴포넌트로 이미지를 최적화하세요.

- [Fonts](https://nextjs.org/docs/pages/getting-started/fonts)
  - 폰트Next.js에서 폰트를 사용하는 방법을 알아보세요.

- [CSS](https://nextjs.org/docs/pages/getting-started/css)
  - CSS Modules, 전역 CSS, Tailwind CSS 등을 포함한 다양한 CSS 추가 방법을 살펴보세요.

- [배포](https://nextjs.org/docs/pages/getting-started/deploying)
  - Next.js 애플리케이션을 배포하는 방법을 알아보세요.
