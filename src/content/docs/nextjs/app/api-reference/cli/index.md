# CLI

최종 업데이트 2026년 2월 20일

Next.js에는 **두 가지** 명령줄 인터페이스(CLI) 도구가 포함되어 있습니다.

  * **`create-next-app`** : 기본 템플릿 또는 공개 GitHub 저장소의 [예제](https://github.com/vercel/next.js/tree/canary/examples)를 사용해 새 Next.js 애플리케이션을 빠르게 생성합니다.
  * **`next`** : Next.js 개발 서버를 실행하고 애플리케이션을 빌드하는 등 다양한 작업을 수행합니다.

- [create-next-app](https://nextjs.org/docs/app/api-reference/cli/create-next-app)
  - create-next-app CLI로 한 번의 명령으로 Next.js 앱을 만드세요.

- [next CLI](https://nextjs.org/docs/app/api-reference/cli/next)
  - Next.js CLI로 애플리케이션을 실행하고 빌드하는 방법을 알아보세요.
