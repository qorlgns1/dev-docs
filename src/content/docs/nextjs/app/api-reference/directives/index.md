# API 레퍼런스: Directives | Next.js
출처 URL: https://nextjs.org/docs/app/api-reference/directives

# 디렉티브

마지막 업데이트 2026년 2월 20일

사용 가능한 디렉티브는 다음과 같습니다:

- [use cache](https://nextjs.org/docs/app/api-reference/directives/use-cache)
  - Next.js 애플리케이션에서 데이터를 캐시하기 위해 "use cache" 디렉티브를 사용하는 방법을 알아보세요.
- [use cache: private](https://nextjs.org/docs/app/api-reference/directives/use-cache-private)
  - 실행 시 요청 API에 접근하는 함수들을 캐시하기 위해 "use cache: private" 디렉티브를 사용하는 방법을 알아보세요.
- [use cache: remote](https://nextjs.org/docs/app/api-reference/directives/use-cache-remote)
  - 원격 캐시 핸들러를 사용해 지속적이고 공유되는 캐싱을 제공하는 "use cache: remote" 디렉티브를 사용하는 방법을 알아보세요.
- [use client](https://nextjs.org/docs/app/api-reference/directives/use-client)
  - use client 디렉티브를 사용해 클라이언트에서 컴포넌트를 렌더링하는 방법을 알아보세요.
- [use server](https://nextjs.org/docs/app/api-reference/directives/use-server)
  - use server 디렉티브를 사용해 서버에서 코드를 실행하는 방법을 알아보세요.
