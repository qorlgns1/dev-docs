# 구성 요소

마지막 업데이트 2026년 2월 20일

- [폰트](https://nextjs.org/docs/app/api-reference/components/font)
  - Font내장된 `next/font` 로더로 웹 폰트 로딩을 최적화하기.

- [폼 컴포넌트](https://nextjs.org/docs/app/api-reference/components/form)
  - Form Component클라이언트 사이드 내비게이션을 사용해 `<Form>` 컴포넌트로 폼 제출과 검색 파라미터 업데이트를 처리하는 방법 학습.

- [이미지 컴포넌트](https://nextjs.org/docs/app/api-reference/components/image)
  - Image Component내장 `next/image` 컴포넌트를 사용해 Next.js 애플리케이션에서 이미지를 최적화하기.

- [링크 컴포넌트](https://nextjs.org/docs/app/api-reference/components/link)
  - Link Component내장 `next/link` 컴포넌트로 빠른 클라이언트 사이드 내비게이션 활성화.

- [스크립트 컴포넌트](https://nextjs.org/docs/app/api-reference/components/script)
  - Script Component내장 `next/script` 컴포넌트를 사용해 Next.js 애플리케이션의 서드파티 스크립트를 최적화하기.
