# 시작하기

최종 업데이트 2026년 2월 20일

Next.js 문서에 오신 것을 환영합니다!

이 **시작하기** 섹션은 첫 Next.js 앱을 만들고 모든 프로젝트에서 활용하게 될 핵심 기능을 익히도록 도와줍니다.

## 사전 지식[](https://nextjs.org/docs/app/getting-started#pre-requisite-knowledge)

이 문서는 웹 개발에 어느 정도 익숙하다고 가정합니다. 시작하기 전에 다음 내용에 익숙하면 도움이 됩니다:

  * HTML
  * CSS
  * JavaScript
  * React



React가 처음이거나 복습이 필요하다면, [React Foundations 과정](https://nextjs.org/learn/react-foundations)과 학습과 동시에 애플리케이션을 만들어 보는 [Next.js Foundations 과정](https://nextjs.org/learn/dashboard-app)을 먼저 살펴보세요.
