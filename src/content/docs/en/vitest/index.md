# Next Generation Testing Framework

A Vite-native testing framework. It's fast!

![Vitest terminal](https://vitest.dev/assets/hero-terminal.C33Vbobj.png)

## Why Vitest

### The Vite Native Test Runner

[Learn more](https://vitest.dev/guide/why)

Vitest was created to make testing just work for [`Vite`](https://vite.dev) apps. By building on top of Vite, Vitest natively understands your Vite config and is able to reuse the same resolve and transform pipelines.

You can also use Vitest even if you are not using Vite. It is Jest-compatible and works for backend code too.

[Learn more](https://vitest.dev/guide/why)

## Fast. Lightweight. Integrated.

##### Vite Powered

Reuse `Vite`'s config and plugins - consistent across your app and tests. But it's not required to use Vitest!

##### Jest Compatible

Expect, snapshot, coverage, and more - migrating from `Jest` is straightforward.

![jest compatible](https://vitest.dev/assets/vitest-jest-compatible.CMjNhpKK.png)

##### Smart & instant watch mode

Only rerun the related changes, just like HMR for tests!

![typed api](https://vitest.dev/assets/vitest-watcher.z485_8FI.png)

##### ESM, TypeScript, JSX

Out-of-box ESM, TypeScript and JSX support powered by `Oxc`.

[See full features list](https://vitest.dev/guide/features)
