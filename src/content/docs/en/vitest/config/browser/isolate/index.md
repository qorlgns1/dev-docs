# browser.isolate

- **Type:** `boolean`
- **Default:** the same as [`--isolate`](https://vitest.dev/config/#isolate)
- **CLI:** `--browser.isolate`, `--browser.isolate=false`

Run every test in a separate iframe.

::: danger DEPRECATED
This option is deprecated. Use [`isolate`](https://vitest.dev/config/#isolate) instead.
:::
