# snapshotSerializers

- **Type:** `string[]`
- **Default:** `[]`

A list of paths to snapshot serializer modules for snapshot testing, useful if you want add custom snapshot serializers. See [Custom Serializer](https://vitest.dev/guide/snapshot#custom-serializer) for more information.
