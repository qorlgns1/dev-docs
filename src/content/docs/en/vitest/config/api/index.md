# api

- **Type:** `boolean | number`
- **Default:** `false`
- **CLI:** `--api`, `--api.port`, `--api.host`, `--api.strictPort`

Listen to port and serve API for [the UI](https://vitest.dev/guide/ui) or [browser server](https://vitest.dev/guide/browser/). When set to `true`, the default port is `51204`.
