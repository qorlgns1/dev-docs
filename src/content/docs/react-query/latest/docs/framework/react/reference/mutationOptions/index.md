# mutationOptions

```tsx
mutationOptions({
  mutationFn,
  ...options,
})
```

**옵션**

일반적으로 [`useMutation`](https://tanstack.com/query/latest/docs/framework/react/reference/useMutation.md)에 전달할 수 있는 모든 값을 `mutationOptions`에도 전달할 수 있습니다.
