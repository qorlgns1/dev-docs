# Codex IDE 확장 기능 슬래시 명령

슬래시 명령을 사용하면 채팅 입력을 벗어나지 않고 Codex를 제어할 수 있습니다. 상태 확인, 로컬/클라우드 모드 전환, 피드백 전송 등에 활용하세요.

## 슬래시 명령 사용하기

1. Codex 채팅 입력란에 `/`를 입력합니다.
2. 목록에서 명령을 선택하거나 계속 입력하여 필터링합니다(예: `/status`).
3. **Enter**를 누릅니다.

## 사용 가능한 슬래시 명령

| 슬래시 명령           | 설명                                                                                  |
| -------------------- | ------------------------------------------------------------------------------------- |
| `/auto-context`      | 최근 파일과 IDE 컨텍스트를 자동으로 포함하도록 자동 컨텍스트를 켜거나 끕니다.            |
| `/cloud`             | 작업을 원격으로 실행하려면 클라우드 모드로 전환합니다(클라우드 액세스 필요).              |
| `/cloud-environment` | 사용할 클라우드 환경을 선택합니다(클라우드 모드에서만 사용 가능).                        |
| `/feedback`          | 피드백 대화 상자를 열어 피드백을 제출하고 로그를 포함할 수도 있습니다.                 |
| `/local`             | 워크스페이스에서 작업을 실행하려면 로컬 모드로 전환합니다.                               |
| `/review`            | 코드 리뷰 모드를 시작하여 커밋되지 않은 변경 사항을 검토하거나 기준 브랜치와 비교합니다. |
| `/status`            | 스레드 ID, 컨텍스트 사용량, 속도 제한을 표시합니다.                                    |
