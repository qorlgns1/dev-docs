# Codex 앱

소스 URL: https://developers.openai.com/codex/app

Codex 앱은 Codex 스레드를 병렬로 작업할 수 있도록 설계된 집중형 데스크톱 환경으로, 내장 worktree 지원, 자동화, 그리고 Git 기능을 제공합니다.

ChatGPT Plus, Pro, Business, Edu, Enterprise 플랜에는 Codex가 포함됩니다. [포함 사항](https://developers.openai.com/codex/pricing)에서 자세히 알아보세요.

## 시작하기

Codex 앱은 macOS(Apple Silicon)에서 사용할 수 있습니다.

  1. Codex 앱 다운로드 및 설치

현재 Codex 앱은 macOS에서만 제공됩니다.

[ macOS용 다운로드 ](https://persistent.oaistatic.com/codex-app-prod/Codex.dmg)

[Windows 및 Linux 알림 받기](https://openai.com/form/codex-app/)

  2. Codex 열기 및 로그인

Codex 앱을 다운로드하고 설치한 후, 앱을 열어 ChatGPT 계정 또는 OpenAI API 키로 로그인하세요.

OpenAI API 키로 로그인하면 [cloud threads](https://developers.openai.com/codex/prompting#threads) 같은 일부 기능을 사용할 수 없을 수 있습니다.

  3. 프로젝트 선택

Codex가 작업할 프로젝트 폴더를 선택하세요.




이전에 Codex 앱, CLI, 또는 IDE Extension을 사용한 적이 있다면, 과거에 작업한 프로젝트가 표시됩니다.

  4. 첫 메시지 보내기

프로젝트를 선택한 뒤 Codex가 로컬 머신에서 작업하도록 `**Local**`이 선택되어 있는지 확인하고, Codex에 첫 메시지를 보내세요.

프로젝트나 컴퓨터 전반에 대해 Codex에 무엇이든 물어볼 수 있습니다. 다음은 몇 가지 예시입니다.

더 많은 아이디어가 필요하면 [탐색 섹션](https://developers.openai.com/codex/explore)을 확인하세요.
