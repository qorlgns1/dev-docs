# Codex

원문 URL: https://developers.openai.com/codex

Codex는 소프트웨어 개발을 위한 OpenAI의 코딩 에이전트입니다. ChatGPT Plus, Pro, Business, Edu, Enterprise 플랜에는 Codex가 포함되어 있습니다. Codex는 다음을 도와줍니다:

  * **코드 작성** : 만들고 싶은 것을 설명하면, Codex가 의도에 맞는 코드를 생성하고 기존 프로젝트 구조와 규약에 맞게 조정합니다.

  * **낯선 코드베이스 이해** : Codex는 복잡하거나 레거시 코드를 읽고 설명하여 팀이 시스템을 어떻게 구성했는지 파악하도록 도와줍니다.

  * **코드 검토** : Codex는 잠재적 버그, 논리 오류, 처리되지 않은 엣지 케이스를 식별하기 위해 코드를 분석합니다.

  * **문제 디버깅 및 수정** : 문제가 발생하면 Codex가 실패를 추적하고 근본 원인을 진단하며, 목표에 맞는 수정안을 제안합니다.

  * **개발 작업 자동화** : Codex는 리팩터링, 테스트, 마이그레이션, 설정 작업 같은 반복 워크플로를 실행해 더 높은 수준의 엔지니어링 작업에 집중할 수 있게 합니다.
