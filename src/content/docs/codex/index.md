# Codex

Codex는 소프트웨어 개발을 위한 OpenAI의 코딩 에이전트입니다. ChatGPT Plus, Pro, Business, Edu, Enterprise 플랜에는 Codex가 포함됩니다. 다음과 같은 도움을 받을 수 있습니다:

  * **코드 작성** : 만들고자 하는 것을 설명하면 Codex가 현재 프로젝트 구조와 컨벤션에 맞추어 의도에 부합하는 코드를 생성합니다.

  * **익숙하지 않은 코드베이스 이해** : Codex는 복잡하거나 레거시 코드를 읽고 설명해 줄 수 있어 팀이 시스템을 어떻게 구성하는지 파악을 돕습니다.

  * **코드 리뷰** : Codex는 잠재적인 버그, 논리 오류, 처리되지 않은 예외 케이스를 파악하기 위해 코드를 분석합니다.

  * **디버그 및 문제 해결** : 문제가 발생하면 Codex가 실패를 추적하고 근본 원인을 진단하며, 목표 지향적인 수정사항을 제안합니다.

  * **개발 업무 자동화** : Codex는 리팩터링, 테스트, 마이그레이션, 설정 작업과 같은 반복적인 워크플로를 실행하여 상위 수준의 엔지니어링 작업에 집중할 수 있게 합니다.
