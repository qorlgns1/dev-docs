# Codex 가격

출처 URL: https://developers.openai.com/codex/pricing

한정된 기간 동안 **ChatGPT Free와 Go에서 Codex를 무료로 체험**하거나, Plus·Pro·Business·Enterprise 구독으로 **2배 Codex 속도 제한**을 누려 보세요.

### Plus

매주 몇 차례 집중된 코딩 세션을 수행하기에 적합합니다.

$20/월

[Plus 가입하기](https://chatgpt.com/explore/plus?utm_internal_source=openai_developers_codex)

  * 웹, CLI, IDE 확장 프로그램, iOS에서 Codex 사용
  * 자동 코드 리뷰·Slack 통합과 같은 클라우드 기반 통합
  * GPT-5.3-Codex를 포함한 최신 모델
  * 로컬 메시지에서 최대 4배 더 높은 사용 한도를 제공하는 GPT-5.1-Codex-Mini
  * [ChatGPT 크레딧](https://developers.openai.com/codex/pricing#credits-overview)으로 유연하게 사용량 확장
  * Plus 플랜에 포함된 기타 [ChatGPT 기능](https://chatgpt.com/pricing)



### Pro

매일 풀타임 개발에 Codex를 활용할 때 적합합니다.

$200/월

[Pro 가입하기](https://chatgpt.com/explore/pro?utm_internal_source=openai_developers_codex)

Plus에 포함된 모든 기능과 다음 혜택:

  * 우선순위가 부여된 요청 처리
  * 일상 코딩 작업을 위한 빠른 Codex 모델 GPT-5.3-Codex-Spark(리서치 미리보기)에 대한 액세스
  * 로컬 및 클라우드 작업에서 6배 더 높은 사용 한도
  * 클라우드 기반 코드 리뷰 10배 증대
  * Pro 플랜에 포함된 기타 [ChatGPT 기능](https://chatgpt.com/pricing)



### Business

스타트업이나 성장 중인 비즈니스에 Codex를 도입하세요.

$30/사용자/월

[무료로 체험](https://chatgpt.com/team-sign-up?utm_internal_source=openai_developers_codex)

Plus에 포함된 모든 기능과 다음 혜택:

  * 클라우드 작업을 더 빠르게 실행하는 대형 가상 머신
  * [ChatGPT 크레딧](https://developers.openai.com/codex/pricing#credits-overview)으로 유연한 사용량 확장
  * 핵심 관리자 제어, SAML SSO, MFA를 갖춘 안전한 전용 작업 공간
  * 기본적으로 비즈니스 데이터로 학습하지 않음. [자세히 알아보기](https://openai.com/business-data/)
  * Business 플랜에 포함된 기타 [ChatGPT 기능](https://chatgpt.com/pricing)



### Enterprise & Edu

엔터프라이즈급 기능으로 조직 전체에 Codex를 제공하세요.

[영업팀 문의](https://chatgpt.com/contact-sales?utm_internal_source=openai_developers_codex)

Business에 포함된 모든 기능과 다음 혜택:

  * 우선순위가 부여된 요청 처리
  * SCIM, EKM, 사용자 분석, 도메인 인증, 역할 기반 접근 제어([RBAC](https://help.openai.com/en/articles/11750701-rbac))를 포함하는 엔터프라이즈 수준 보안 및 제어
  * [Compliance API](https://chatgpt.com/admin/api-reference#tag/Codex-Tasks)를 통한 감사 로그와 사용량 모니터링
  * 데이터 보존 및 데이터 거주지 제어
  * Enterprise 플랜에 포함된 기타 [ChatGPT 기능](https://chatgpt.com/pricing)



### API Key

CI와 같은 공유 환경에서 자동화에 적합합니다.

[자세히 알아보기](https://developers.openai.com/codex/auth)

  * CLI, SDK, IDE 확장에서 Codex 사용
  * 클라우드 기반 기능 없음(GitHub 코드 리뷰, Slack 등)
  * GPT-5.3-Codex와 GPT-5.3-Codex-Spark 같은 신규 모델에는 지연된 액세스
  * [API 가격](https://platform.openai.com/docs/pricing)에 따라 Codex가 사용한 토큰에 대해서만 비용 지불



## 자주 묻는 질문

### 내 플랜의 사용 한도는 무엇인가요?

보낼 수 있는 Codex 메시지 수는 코딩 작업의 규모·복잡성과 로컬 실행인지 클라우드 실행인지에 따라 달라집니다. 작은 스크립트나 일상적인 함수는 할당량의 일부만 사용하지만, 대형 코드베이스·장시간 실행 작업·더 많은 컨텍스트가 필요한 긴 세션은 메시지당 훨씬 더 많은 한도를 소모합니다.

| 로컬 메시지[*](https://developers.openai.com/codex/pricing#shared-limits) / 5h| 클라우드 작업[*](https://developers.openai.com/codex/pricing#shared-limits) / 5h| 주간 코드 리뷰  
---|---|---|---  
ChatGPT Plus| 45-225| 10-60| 10-25  
ChatGPT Pro| 300-1500| 50-400| 100-250  
ChatGPT Business| 45-225| 10-60| 10-25

ChatGPT Enterprise & Edu| 고정 한도 없음 — 사용량은 [credits](https://developers.openai.com/codex/pricing#credits-overview)에 따라 확장됩니다  
API Key| [사용량 기반](https://platform.openai.com/docs/pricing)| 사용 불가| 사용 불가  
  
*로컬 메시지와 클라우드 작업의 사용 한도는 동일한 **5시간 창**을 공유합니다. 추가 주간 한도가 적용될 수 있습니다.

유연한 요금제가 없는 Enterprise 및 Edu 플랜은 대부분의 기능에서 Plus와 동일한 좌석당 사용 한도를 가집니다.

GPT-5.1-Codex-Mini는 로컬 작업에 사용할 수 있으며 최대 4배 더 많은 사용량을 제공합니다.

GPT-5.3-Codex-Spark는 ChatGPT Pro 사용자 전용 연구 프리뷰이며, 출시 시점에는 API에서 제공되지 않습니다. 이 모델은 특수한 저지연 하드웨어에서 실행되므로, 사용량은 수요에 따라 조정될 수 있는 별도 사용 한도로 관리됩니다.

### 사용 한도에 도달하면 어떻게 되나요?

사용 한도에 도달한 ChatGPT Plus와 Pro 사용자는 기존 플랜을 업그레이드하지 않고도 추가 크레딧을 구매해 작업을 계속할 수 있습니다.

[flexible pricing](https://help.openai.com/en/articles/11487671-flexible-pricing-for-the-enterprise-edu-and-business-plans)이 적용된 Business, Edu, Enterprise 플랜도 추가 워크스페이스 크레딧을 구매해 Codex를 계속 사용할 수 있습니다.

사용 한도에 가까워지면 GPT-5.1-Codex-Mini 모델로 전환해 한도를 더 오래 유지할 수 있습니다.

모든 사용자는 API key를 사용해 추가 로컬 작업을 실행할 수 있으며, 사용량은 [standard API rates](https://platform.openai.com/docs/pricing)에 따라 청구됩니다.

### 현재 사용 한도를 어디서 확인할 수 있나요?

현재 한도는 [Codex usage dashboard](https://chatgpt.com/codex/settings/usage)에서 확인할 수 있습니다. 활성 Codex CLI 세션 중 남은 한도를 확인하려면 `/status`를 사용할 수 있습니다.

### 크레딧은 어떻게 작동하나요?

크레딧은 포함된 사용 한도에 도달한 이후에도 Codex를 계속 사용할 수 있게 해 줍니다. 사용한 모델과 기능에 따라 사용량이 크레딧에서 차감되어 중단 없이 작업을 연장할 수 있습니다.

메시지당 크레딧 비용은 작업 규모, 복잡도, 필요한 추론 수준에 따라 달라집니다. 아래 표는 평균 크레딧 비용을 보여 주며, 이 평균은 기존 GPT-5.2, GPT-5.2-Codex, GPT-5.1, GPT-5.1-Codex-Max, GPT-5, GPT-5-Codex, GPT-5-Codex-Mini에도 동일하게 적용됩니다. 새로운 기능이 도입되면 평균 요율이 변경될 수 있습니다.

| 단위| GPT-5.3-Codex, GPT-5.2-Codex| GPT-5.1-Codex-Mini  
---|---|---|---  
로컬 작업| 1 message| ~5 credits| ~1 credit  
클라우드 작업| 1 message| ~25 credits| 사용 불가  
코드 리뷰| 1 pull request| ~25 credits| 사용 불가  
  
[ChatGPT Plus 및 Pro에서 크레딧을 사용하는 방법 자세히 보기.](https://help.openai.com/en/articles/12642688-using-credits-for-flexible-usage-in-chatgpt-freegopluspro-sora)  
[ChatGPT Business, Enterprise, Edu에서 크레딧을 사용하는 방법 자세히 보기.](https://help.openai.com/en/articles/11487671-flexible-pricing-for-the-enterprise-edu-and-business-plans)

### 코드 리뷰 사용량에는 무엇이 포함되나요?

코드 리뷰 사용량은 Codex가 GitHub를 통해 리뷰를 실행할 때만 적용됩니다. 예를 들어 pull request에서 `@Codex`를 태그하거나 저장소에서 자동 리뷰를 활성화한 경우입니다. 로컬이나 GitHub 외부에서 실행되는 리뷰는 일반 사용 한도에 포함됩니다.

### 사용 한도를 더 오래 유지하려면 무엇을 할 수 있나요?

위에 제시한 사용 한도와 크레딧은 평균 요율입니다. 다음 팁을 통해 한도를 최대화해 보세요:

  * **프롬프트 크기를 제어하세요.** Codex에 제공하는 지시는 명확하게 작성하되 불필요한 맥락은 제거하세요.
  * **AGENTS.md의 크기를 줄이세요.** 대규모 프로젝트에서 작업할 경우 [저장소 내에 중첩](https://developers.openai.com/codex/guides/agents-md#layer-project-instructions)해 AGENTS.md를 통해 주입하는 컨텍스트 양을 조절할 수 있습니다.
  * **사용하는 MCP 서버 수를 제한하세요.** Codex에 추가하는 각 [MCP](https://developers.openai.com/codex/mcp)는 메시지에 더 많은 컨텍스트를 추가하며, 그만큼 한도를 더 많이 사용합니다. 필요하지 않을 때는 MCP 서버를 비활성화하세요.

* **일상적인 작업에서는 GPT-5.1-Codex-Mini로 전환하세요.** 미니 모델을 사용하면 사용 한도가 약 4배 늘어납니다.
