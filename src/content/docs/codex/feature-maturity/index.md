# 기능 성숙도

출처 URL: https://developers.openai.com/codex/feature-maturity

일부 Codex 기능은 신뢰도, 향후 변경 가능성, 기대할 수 있는 지원 수준을 이해할 수 있도록 성숙도 라벨과 함께 제공됩니다.

성숙도|의미|안내  
---|---|---  
개발 중|아직 사용할 준비가 되지 않았습니다.|사용하지 마세요.  
실험적|불안정하며 OpenAI가 제거하거나 변경할 수 있습니다.|위험을 감수하고 사용하세요.  
베타|광범위한 테스트가 가능하며 대부분 완성되었으나, 사용자 피드백에 따라 일부가 달라질 수 있습니다.|대부분의 평가 및 파일럿에 적합하며 작은 변경 가능성을 예상하세요.  
안정|완전히 지원되고 문서화되어 있으며 폭넓은 사용에 준비되어 있습니다. 동작과 설정이 시간이 지나도 일관됩니다.|프로덕션 사용에 안전하며, 제거가 필요할 때는 일반적으로 사용 중단 절차를 거칩니다.
