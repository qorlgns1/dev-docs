# 기능 성숙도

일부 Codex 기능에는 성숙도 레이블이 붙어 있어 각 기능의 신뢰도, 변경 가능성, 기대할 수 있는 지원 수준을 파악할 수 있습니다.

| 성숙도               | 의미                                                                                                           | 안내 사항                                                                       |
| -------------------- | -------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| 개발 중              | 아직 사용할 준비가 되어 있지 않습니다.                                                                          | 사용하지 마세요.                                                                 |
| 실험적               | 불안정하며 OpenAI가 삭제하거나 변경할 수 있습니다.                                                              | 위험을 감수하고 사용하세요.                                                      |
| 베타                 | 폭넓은 테스트를 위한 준비가 되어 있으며 대부분 완성되었지만 사용자 피드백에 따라 일부 요소는 변경될 수 있습니다. | 대부분 평가나 시범 운영에는 적합하며, 소규모 변경을 예상하세요.                 |
| 안정적               | 완전히 지원되고 문서화되어 있으며 폭넓게 사용할 준비가 되어 있습니다. 행동 및 설정이 시간에 따라 일관되게 유지됩니다. | 프로덕션 환경에 안전하게 사용할 수 있으며, 제거 시에는 일반적으로 사용 중단 절차를 거칩니다. |
