# Codex

![Codex 앱이 프로젝트 사이드바, 스레드 목록, 리뷰 창을 보여주는 모습](https://developers.openai.com/images/codex/app/codex-app-basic-light.webp)

Codex는 소프트웨어 개발을 위한 OpenAI의 코딩 에이전트입니다. ChatGPT Plus, Pro, Business, Edu, Enterprise 플랜에는 Codex가 포함되어 있습니다. Codex가 도와줄 수 있는 일은 다음과 같습니다:

- **코드 작성**: 만들고자 하는 내용을 설명하면 Codex가 기존 프로젝트 구조와 관례에 맞춰 의도에 부합하는 코드를 생성합니다.

- **낯선 코드베이스 이해**: Codex는 복잡하거나 레거시 코드도 읽고 설명해주므로 팀이 시스템을 어떻게 구성했는지 파악할 수 있습니다.

- **코드 리뷰**: Codex는 잠재적 버그, 논리 오류, 처리되지 않은 엣지 케이스를 파악하기 위해 코드를 분석합니다.

- **디버그 및 문제 해결**: 문제가 발생하면 Codex가 실패를 추적하고 근본 원인을 진단하며 구체적인 수정 방향을 제안합니다.

- **개발 작업 자동화**: Codex는 리팩터링, 테스트, 마이그레이션, 설정 작업처럼 반복적인 워크플로를 실행하여 더 높은 수준의 엔지니어링 업무에 집중할 수 있도록 합니다.
