# 오픈 소스

Source URL: https://developers.openai.com/codex/open-source

OpenAI는 Codex의 핵심 부분을 공개적으로 개발합니다. 해당 작업은 GitHub에 공개되어 있어 진행 상황을 따르고, 이슈를 보고하고, 개선 사항을 기여할 수 있습니다.

## 오픈 소스 구성 요소

Component| Where to find| Notes  
---|---|---  
Codex CLI| [openai/codex](https://github.com/openai/codex)| Codex 오픈 소스 개발의 주요 거점  
Codex SDK| [openai/codex/sdk](https://github.com/openai/codex/tree/main/sdk)| SDK 소스는 Codex 리포지토리에 포함  
Codex App Server| [openai/codex/codex-rs/app-server](https://github.com/openai/codex/tree/main/codex-rs/app-server)| 앱 서버 소스는 Codex 리포지토리에 포함  
Skills| [openai/skills](https://github.com/openai/skills)| Codex를 확장하는 재사용 가능한 스킬  
IDE extension| -| 오픈 소스 아님  
Codex web| -| 오픈 소스 아님  
Universal cloud environment| [openai/codex-universal](https://github.com/openai/codex-universal)| Codex 클라우드에서 사용하는 기본 환경  
  
## 이슈 보고 및 기능 요청 위치

Codex 구성 요소 전반에 대한 버그 보고와 기능 요청은 Codex GitHub 리포지토리를 사용하세요.

  * 버그 보고 및 기능 요청: [openai/codex/issues](https://github.com/openai/codex/issues)
  * 토론 포럼: [openai/codex/discussions](https://github.com/openai/codex/discussions)

이슈를 등록할 때는 가능하면 사용 중인 구성 요소(CLI, SDK, IDE extension, Codex web)와 버전을 함께 포함해 주세요.
