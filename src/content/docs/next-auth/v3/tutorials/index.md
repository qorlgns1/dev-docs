### [Next.js와 NextAuth.js를 사용한 풀스택 인증 예제](https://github.com/prisma/prisma-examples/tree/latest/typescript/rest-nextjs-api-routes-auth)[​](https://next-auth.js.org/v3/tutorials#fullstack-authentication-example-with-nextjs-and-nextauthjs "Direct link to heading")

### [NextAuth.js 소개](https://www.youtube.com/watch?v=npZsJxWntJM)[​](https://next-auth.js.org/v3/tutorials#introduction-to-nextauthjs "Direct link to heading")
