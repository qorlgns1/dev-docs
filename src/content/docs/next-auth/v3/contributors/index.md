## 역사[​](https://next-auth.js.org/v3/contributors#history "헤딩으로 직접 링크")

NextAuth.js는 2016년에 [Iain Collins](https://github.com/iaincollins)가 처음 개발했습니다.

2020년에 NextAuth.js는 Serverless를 지원하도록 처음부터 다시 구축되었고, MySQL, Postgres, MongoDB 지원, JSON Web Tokens, 그리고 12개가 넘는 인증 provider에 대한 내장 지원이 추가되었습니다.

2025년에 NextAuth.js는 2016년에 [Better Auth](https://better-auth.com)에 합류했습니다.
