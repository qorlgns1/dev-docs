### [NextAuth와 Fauna를 사용한 Next.js 매직 링크 인증](https://alterclass.io/tutorials/magic-link-authentication-in-nextjs-with-nextauth-and-fauna) [​](https://next-auth.js.org/tutorials#magic-link-authentication-in-nextjs-with-nextauth-and-fauna-external-- "Direct link to heading")

NextAuth와 Fauna DB를 사용해 Next.js 프로젝트에서 데이터베이스 저장 기반 비밀번호 없는/매직 링크 인증을 구현하는 방법을 배워보세요.

> 프로젝트 코드의 최종 버전은 [Github](https://github.com/AlterClassIO/magic-next-auth)에서 확인할 수 있습니다. 비밀번호 없는 인증이 필요한 모든 Next.js 앱의 시작점으로 사용할 수 있습니다.

이 튜토리얼에서 다루는 내용:

- Next.js, NextAuth.js, Fauna를 매끄럽게 함께 동작하도록 구성하기
- 인증 요청 처리를 위해 Next.js 동적 API routes 사용하기
- Fauna와 `next-auth`용 Fauna Adapter를 사용해 사용자, 이메일 로그인 토큰, 세션 영속화하기
- React + Tailwind CSS로 커스텀 로그인 및 확인 페이지 만들기
- 로그인 이메일 커스터마이징 및 신규 사용자 환영 이메일 발송하기
