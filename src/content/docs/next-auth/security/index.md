# 보안 | NextAuth.js

버전: v4

## 취약점 제보[​](https://next-auth.js.org/security#reporting-a-vulnerability "Direct link to heading")

NextAuth.js는 책임 있는 공개 원칙을 따릅니다.

NextAuth.js를 사용하는 사이트의 보안에 영향을 줄 수 있는 심각한 문제를 제보할 때는, 저희에게 직접 연락해 주시길 요청드립니다.

심각한 문제와 관련해 연락해 주시면:

- 72시간 이내에 회신드리기 위해 노력하겠습니다.
- 30일 이내에 수정 사항을 배포하는 것을 목표로 하겠습니다.
- 문제를 해결하는 수정 사항이 배포되면, 해당 이슈를 공개하고(동의해 주신 경우) 기여자를 명시하겠습니다.
- 90일이 경과했는데도 수정 사항이 없으면, 해당 이슈를 공개적으로 공개하겠습니다.

이슈를 제보하는 가장 좋은 방법은 [info@balazsorban.com](mailto:info@balazsorban.com), [hi@thvu.dev](mailto:hi@thvu.dev), [yo@ndo.dev](mailto:yo@ndo.dev)로 이메일을 보내거나, 선호하시는 방식으로 자세한 논의를 위해 연락을 요청하는 공개 이슈를 등록하는 것입니다. (이 단계에서는 민감한 세부 정보를 공개적으로 노출하지 마세요.)

note

덜 심각한 이슈(예: 지원되지 않는 흐름의 RFC 준수 문제 또는 향후 문제가 될 수 있는 잠재적 이슈)의 경우, 이를 공개 버그 리포트나 기능 요청으로 제출하거나, 공개 토론을 열기 위한 질문을 올리는 것이 적절합니다.
