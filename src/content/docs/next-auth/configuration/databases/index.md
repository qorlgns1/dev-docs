# 데이터베이스 | NextAuth.js

버전: v4

NextAuth.js는 여러 데이터베이스 어댑터를 제공합니다. [개요](https://authjs.dev/getting-started/database)를 확인하세요.

> **v4**부터 NextAuth.js는 기본적으로 어댑터를 포함해 제공되지 않습니다. 어떤 정보든 영속적으로 저장하려면, 제공되는 여러 어댑터 중 하나를 직접 설치해야 합니다. 자세한 내용은 각 어댑터 문서 페이지를 참고하세요.

NextAuth.js에서 데이터베이스가 어떻게 사용되는지 더 알아보려면 [FAQ의 데이터베이스 섹션](https://next-auth.js.org/faq#databases)을 확인하세요.

## 데이터베이스 사용 방법[​](https://next-auth.js.org/configuration/databases#how-to-use-a-database "Direct link to heading")

고급 구성에 대한 자세한 내용(예: [custom adapter](https://authjs.dev/guides/creating-a-database-adapter)를 사용해 NextAuth.js를 다른 데이터베이스와 함께 사용하는 방법 포함)은 [어댑터 문서](https://authjs.dev/getting-started/database)를 참고하세요.
