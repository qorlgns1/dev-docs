# 개요 | NextAuth.js

버전: v4

**NextAuth.js**의 인증 프로바이더(Authentication Providers)는 사용자가 로그인할 때 사용할 수 있는 서비스입니다.

사용자가 로그인할 수 있는 방법은 네 가지입니다.

- [내장 OAuth Provider 사용](https://next-auth.js.org/configuration/providers/oauth) (예: Github, Twitter, Google 등...)
- [커스텀 OAuth Provider 사용](https://next-auth.js.org/configuration/providers/oauth#using-a-custom-provider)
- [이메일 사용](https://next-auth.js.org/configuration/providers/email)
- [Credentials 사용](https://next-auth.js.org/configuration/providers/credentials)

note

NextAuth.js는 어떤 OAuth 서비스와도 동작하도록 설계되었으며, **OAuth 1.0**, **1.0A**, **2.0** 및 **OpenID Connect (OIDC)**를 지원하고, 가장 널리 사용되는 로그인 서비스 대부분을 기본 지원합니다.
