### 프레임워크 가이드

다양한 인기 환경에서 Tailwind CSS를 설치할 때 권장하는 접근 방식을 다루는 프레임워크별 가이드입니다.

-

* Next.js

뛰어난 개발자 경험을 제공하는 풀기능 React 프레임워크입니다.

-

* Laravel

표현력이 풍부하고 우아한 문법을 갖춘 PHP 웹 애플리케이션 프레임워크입니다.

-

* Nuxt

범용 애플리케이션 구축을 위한 직관적인 Vue 프레임워크입니다.

-

* SolidJS

단순하고 성능이 뛰어나며 반응형 사용자 인터페이스를 구축하기 위한 도구입니다.

-

* SvelteKit

Svelte.js로 모든 규모의 앱을 가장 빠르게 구축하는 방법입니다.

-

* Gatsby

React와 GraphQL로 정적 사이트를 구축하기 위한 프레임워크입니다.

-

* Angular

모바일 및 데스크톱 웹 애플리케이션을 구축하기 위한 플랫폼입니다.

-

* Ruby on Rails

놀라운 웹 앱을 만드는 데 필요한 모든 도구를 갖춘 풀스택 프레임워크입니다.

-

* React Router

어디에나 배포할 수 있는 표준 중심 라우터입니다.

-

* TanStack Start

React와 Solid를 위한 TanStack Router 기반의 풀스택 프레임워크입니다.

-

* Phoenix

Elixir로 풍부하고 상호작용적인 애플리케이션을 구축하기 위한 프레임워크입니다.

-

* Parcel

웹을 위한 제로 설정 빌드 도구입니다.

-

* Symfony

웹사이트와 웹 애플리케이션을 만들기 위한 PHP 프레임워크입니다.

-

* Meteor

크로스플랫폼 앱 개발을 위한 풀스택 JavaScript 프레임워크입니다.

-

* AdonisJS

Node.js를 위한 완전한 기능의 웹 프레임워크입니다.

-

* Ember.js

야심찬 웹 개발자를 위한 JavaScript 프레임워크입니다.

-

* Astro

속도를 위해 설계된 올인원 웹 프레임워크입니다.

-

* Qwik

별다른 노력 없이 즉시 상호작용하는 웹 앱을 구축하세요.

-

* Rspack

빠른 Rust 기반 웹 번들러입니다.

원하는 프레임워크가 보이지 않나요? 대신 [Tailwind CLI](https://tailwindcss.com/docs/installation/tailwind-cli), [Vite plugin](https://tailwindcss.com/docs/installation/using-vite), 또는 [PostCSS plugin](https://tailwindcss.com/docs/installation/using-postcss)을 사용해 보세요.
